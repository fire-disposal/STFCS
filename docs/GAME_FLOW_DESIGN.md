# STFCS 游戏流程设计文档

## 一、角色系统设计

### 1.1 角色类型

| 角色 | 职责 | 权限 |
|------|------|------|
| **DM (Dungeon Master)** | 游戏主持人，控制敌方单位、管理游戏进程 | 启动部署、创建敌方单位、推进回合、暂停/结束游戏 |
| **玩家** | 控制己方阵营的舰船 | 部署舰船、移动、攻击、使用技能 |
| **观察者** | 观看游戏进程 | 无操作权限 |

### 1.2 阵营系统

每个玩家选择一个阵营进行游戏：

| 阵营 | 描述 | 特色 |
|------|------|------|
| **霸主 (Hegemony)** | 霸主联盟 | 平衡型，重装甲 |
| **辛德里亚 (Sindrian)** | 辛德里亚帝国 | 能量武器优势 |
| **珀尔修斯 (Persean)** | 珀尔修斯联盟 | 导弹优势 |
| **三叠纪 (Tri-Tachyon)** | 三叠纪公司 | 高科技，护盾优势 |
| **海盗 (Pirate)** | 海盗势力 | 快速，低成本 |
| **独立 (Independent)** | 中立势力 | 多样化 |

### 2.2 各阶段详细设计

#### 阶段一：大厅 (Lobby)

**玩家操作**：
1. 输入玩家名称
2. 创建新房间 或 加入已有房间
3. 查看房间列表（显示房间名、玩家数、状态）

**UI元素**：
- 玩家名称输入框
- 创建房间按钮
- 房间列表（可刷新）
- 快速加入按钮

#### 阶段二：房间准备 (Room)

**玩家操作**：
1. 选择角色：DM 或 玩家
2. 如果是玩家，选择阵营
3. 等待其他玩家加入
4. DM可以设置游戏参数（地图、胜利条件等）
5. DM点击"开始游戏"进入部署阶段

**UI元素**：
- 玩家列表（显示名称、角色、阵营、就绪状态）
- 角色选择区（DM/玩家）
- 阵营选择区（仅玩家可见）
- 游戏设置面板（仅DM可见）
- 开始游戏按钮（仅DM可见）
- 离开房间按钮

**权限控制**：
- 每个房间只能有1个DM
- 每个阵营只能有1个玩家
- DM可以踢出玩家
- DM可以设置房间为私密

#### 阶段三：部署 (Deployment)

**DM操作**：
1. 设置各阵营的部署区域
2. 创建敌方单位（可选）
3. 查看玩家部署进度
4. 确认开始战斗

**玩家操作**：
1. 从舰船列表选择要部署的舰船
2. 在己方部署区域内放置舰船
3. 调整舰船朝向
4. 确认部署完成

**UI元素**：
- 地图视图（显示部署区域）
- 舰船选择面板
- 部署点数显示
- 阵营就绪状态
- 部署完成按钮

**引导提示**：
```
📋 部署阶段指南

作为 DM：
1. 设置各阵营的部署区域（拖拽调整）
2. 可选：在地图上放置敌方单位
3. 等待所有玩家完成部署
4. 点击"开始战斗"按钮

作为 玩家：
1. 从左侧选择要部署的舰船
2. 在地图上点击放置位置
3. 使用滚轮调整朝向
4. 点击"确认部署"完成
5. 等待其他玩家和DM
```

#### 阶段四：战斗 (Playing)

**回合流程**：
```

    │
    ▼

    │
    ▼

│          DM行动阶段                 │
│  1. 控制敌方单位移动                │
│  2. 控制敌方单位攻击                │
│ 3. 触发事件                         │
└─────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────┐
│          回合结算                   │
│  1. 辐能消散                        │
```

**UI元素**：
- 战术地图
- 舰船信息面板
- 行动控制面板
- 回合指示器
- 目标/武器选择器

1. **游戏设置**

## 四、玩家引导系统

### 4.1 引导提示类型

1. **阶段引导** - 进入新阶段时显示
2. **操作提示** - 鼠标悬停或选中时显示
3. **错误提示** - 操作失败时显示
4. **状态提示** - 游戏状态变化时显示

### 4.2 引导组件设计

```typescript
interface GameGuide {
  // 当前阶段的引导步骤
  currentPhase: GamePhase;
  steps: GuideStep[];
  
  // 是否显示引导
  showGuide: boolean;
  
  // 当前步骤索引
  currentStep: number;
}

interface GuideStep {
  id: string;
  title: string;
  description: string;
  targetElement?: string; // 高亮的UI元素
  action?: 'click' | 'drag' | 'select'; // 需要执行的操作
  completed: boolean;
}
```

### 4.3 各阶段引导内容

#### 大厅阶段
```
1. 欢迎来到 STFCS！
2. 输入你的名称
3. 创建一个房间或加入已有房间
```

#### 房间阶段
```
作为 DM：
1. 等待玩家加入
2. 确认所有玩家已选择阵营
3. 点击"开始游戏"

作为 玩家：
1. 选择你的阵营
2. 等待DM开始游戏
```

#### 部署阶段
```
1. 从左侧选择舰船
2. 在地图上点击放置
3. 调整朝向
4. 点击"确认部署"
```

#### 战斗阶段
```
1. 选择你的舰船
2. 执行移动（三阶段）
3. 选择武器和目标
4. 执行攻击
5. 结束回合
```
