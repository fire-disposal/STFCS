# STFCS 快速启动指南

## 🚀 初步测试准备完成

本项目已完成核心功能集成，可用于初步测试。

## 📦 启动步骤

### 1. 启动服务端

```bash
cd packages/server
pnpm dev
```

服务端将在 `ws://localhost:2567` 启动

### 2. 启动客户端（新终端）

```bash
cd packages/client
pnpm dev
```

客户端将在 `http://localhost:5173` 启动

### 3. 访问应用

打开浏览器访问 `http://localhost:5173`

### 🎯 测试流程

#### 基础测试
1. **连接测试**
   - 输入玩家名称
   - 点击"连接并加入"
   - 确认成功进入房间

2. **视角控制测试**
   - 测试地图缩放（放大/缩小）
   - 测试地图平移（上下左右）
   - 测试网格开关
   - 打开设置菜单调整坐标精度

3. **舰船选择测试**
   - 点击舰船进行单选
   - 查看舰船详情面板
   - 测试磁性指针吸附效果

#### DM 功能测试
1. **创建舰船**
   - 以 DM 身份登录
   - 打开"对象创建工具"
   - 选择舰船型号
   - 点击地图摆放
   - 指定阵营和归属

2. **舰船控制**
   - 选择创建的舰船
   - 测试移动操作
   - 测试护盾开关
   - 测试武器开火

3. **DM 特殊功能**
   - 清除舰船过载
   - 修改装甲值
   - 分配舰船控制权
   - 强制进入下一阶段

#### 战斗系统测试
1. **移动测试**
   - 选择己方舰船
   - 调整三阶段移动参数
   - 执行移动
   - 验证移动距离限制

2. **开火测试**
   - 选择武器
   - 选择目标
   - 执行开火
   - 查看伤害结果

3. **辐能测试**
   - 开启护盾观察辐能消耗
   - 测试主动排散
   - 触发过载状态

## ⚙️ 设置选项

打开设置菜单（点击顶部栏 ⚙️ 按钮）可调整：

### 通用设置
- **坐标精度**：精确/舍入到 10/舍入到 100
- **角度模式**：度数/航海角度/弧度

### 视图设置
- **网格显示**：显示/隐藏
- **缩放级别**：50%-300%

## 🛠️ 调试技巧

### 查看连接状态
顶部栏显示：
- 连接质量（excellent/good/fair/poor/offline）
- Ping 值（毫秒）

### 查看舰船状态
选择舰船后查看：
- 船体值（HP）
- 装甲值（6 象限）
- 辐能值（软/硬）
- 移动状态
- 武器冷却

### 错误处理
- 连接错误会显示在底部
- 点击"关闭"按钮可清除错误

## 📝 已知限制

1. **多人同步**：需要服务端运行
2. **资源导入**：暂不支持自定义贴图
3. **音频系统**：尚未实现
4. **保存系统**：暂不支持存档

## 🔧 故障排除

### 无法连接服务端
1. 确认服务端已启动（端口 2567）
2. 检查防火墙设置
3. 查看浏览器控制台错误

### 客户端显示异常
1. 清除浏览器缓存
2. 重新构建：`pnpm build`
3. 检查控制台错误信息

### DM 功能不可用
1. 确认当前玩家角色为 DM
2. 第一个连接的玩家自动成为 DM
3. 查看顶部栏玩家列表确认角色
